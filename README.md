# TodoList

**<ins>[В данной работе](https://levineye13.github.io/TodoList/index.html)</ins>** пользователю представлена возможность вносить список своих задач и планов на будущее.
Все элементы страцы являются динамическими, то есть рендерятся с помощью JS, который, в свою очередь, построен на ООП. Сущности имеют между собой
<ins>слабую</ins> связь, а работа с данными вынесена за счет Redux.
